# Phase 8a: HTTP API — Projects + Chargesheet (no slicing yet)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make logos serve `/api/v1/health` + project CRUD + chargesheet download over HTTP on `:7777`, matching the existing `mock-daemon/server.ts` contract closely enough that the SvelteKit chargesheet-ui can create, list, view, and delete projects against the real daemon.

**Architecture:** A small HTTP module (`src/api/`) listens on `:7777`, dispatches requests through a method+path router, and handlers translate between HTTP and the existing `db.projects` CRUD + a thin filesystem layer (`src/storage/project_dir.zig`) that owns `data_dir/<project_id>/chargesheet.pdf`. JSON marshalling is done by hand against the UI's Zod schemas (in `src/lib/api/schemas.ts`). Multipart form-data is parsed by a minimal in-tree parser; we accept the whole upload into RAM (matching the mock). No worker thread or slicing endpoints yet — that's Phase 8b. Existing `main.zig` is rewritten to replace its stdin-EOF block with the HTTP serve loop.

**Tech Stack:** Zig 0.16 stdlib only (`std.http.Server`, `std.Io.net`, `std.json`, `std.crypto.random`), the existing logos `db.*` modules, `mupdf-zig` (for `pageCount` during upload validation).

**Scope (this plan, Phase 8a):**

| Endpoint | Method | Purpose |
|---|---|---|
| `/api/v1/health` | GET | smoke endpoint, returns `{status:"ok",version:"..."}` |
| `/api/v1/projects` | GET | list all projects, newest-opened first |
| `/api/v1/projects` | POST | multipart upload — create a project from a PDF |
| `/api/v1/projects/:id` | GET | fetch one project, touch `last_opened_at` |
| `/api/v1/projects/:id` | DELETE | remove project + cascade slices/jobs + rmdir |
| `/api/v1/projects/:id/chargesheet` | GET | stream the uploaded PDF back to the UI |
| `/api/v1/*` | OPTIONS | CORS preflight |

**Out of scope** (Phase 8b/c):
- `POST /api/v1/projects/:id/jobs/slice`
- `GET /api/v1/projects/:id/jobs/:job_id`
- `GET /api/v1/projects/:id/slices`
- `DELETE /api/v1/projects/:id/slices/:filename`
- `GET /api/v1/projects/:id/slices/:filename`
- Worker thread / background slicing

**The UI's JSON contract** (from `chargesheet-ui/src/lib/api/schemas.ts`) — every response must match these shapes:

```json
// Project (returned by POST /projects, GET /projects/:id; element of GET /projects array)
{
  "id": "proj_<uuid-with-dashes>",
  "name": "string",
  "description": "string | null",
  "created_at": "2026-05-25T12:34:56Z",
  "last_opened_at": "2026-05-25T12:34:56Z",
  "chargesheet": { "filename": "string", "page_count": <int>, "size_bytes": <int> }
}

// Error (any non-2xx response)
{ "code": "ERROR_CODE", "message": "human-readable", "details": <any | omitted> }

// Health
{ "status": "ok", "version": "0.1.0" }
```

The DB stores the chargesheet fields flat (`chargesheet_filename`, `chargesheet_page_count`, `chargesheet_size_bytes`); the JSON nests them under `chargesheet`. The JSON serializer does this transformation.

## File Structure

```
src/
├── api/
│   ├── server.zig         # bind + accept-loop + per-connection handling; exports `pub fn serve(...)`
│   ├── router.zig         # method+path matching with `:id` extraction; pure function
│   ├── handlers.zig       # all endpoint handler functions
│   ├── json.zig           # JSON encoders for Project, ApiError, Health
│   └── multipart.zig      # minimal multipart/form-data parser (boundary-delimited fields)
├── storage/
│   └── project_dir.zig    # `data_dir/<project_id>/chargesheet.pdf` layout + read/write helpers
├── ids.zig                # `proj_<uuid>` ID generation via std.crypto.random
├── main.zig               # MODIFIED: replace stdin-block with api.serve(...)
└── (existing src/db/* and other modules unchanged)
```

Conventions:
- All HTTP responses encode JSON via `src/api/json.zig` helpers — never hand-write JSON in handlers.
- Handlers take `(gpa, io, db, request, params)` and return a small `Response` struct or write directly to the request's response stream.
- Errors map to `{code, message, details?}` JSON via a single `respondError(req, status, code, message)` helper.
- Filesystem paths via `std.fs.path.join` — no string concatenation.
- All IDs generated by `ids.generateProjectId(gpa)` — single source of truth.
- CORS headers set for every `/api/*` response, including preflight `OPTIONS`. Origin matched against `http://localhost:5173`.

**Critical Zig 0.16 caveats** (carry-over from prior phases):
- `std.Io.Dir.cwd()` not `std.fs.cwd()`.
- File reads use `file.readPositionalAll(io, buf, 0)` or `file.reader(io, buf)`.
- `std.http.Server` is a per-connection state machine (see `lib/std/http/Server.zig:46 pub fn receiveHead(s: *Server) ReceiveHeadError!Request`). The implementer MUST read that file and the test file at `lib/std/http/test.zig` to find the correct accept-loop + reader/writer wiring pattern. **Do not synthesize the HTTP API from memory or from training data — read the source.**

```zig
const std = @import("std");
const testing = std.testing;

pub const Method = enum { GET, POST, DELETE, OPTIONS };

pub const Route = enum {
    health,
    projects_list,
    projects_create,
    projects_get,
    projects_delete,
    projects_chargesheet,
    not_found,
    cors_preflight,
};

pub const Match = struct {
    route: Route,
    /// For routes with `:id`, the extracted value (slice into the original path).
    id: ?[]const u8,
};

/// Pure function: classify a method+path into a Route + optional :id extraction.
pub fn match(method: Method, path: []const u8) Match {
    if (method == .OPTIONS and std.mem.startsWith(u8, path, "/api/")) {
        return .{ .route = .cors_preflight, .id = null };
    }
    if (method == .GET and std.mem.eql(u8, path, "/api/v1/health")) return .{ .route = .health, .id = null };
    if (method == .GET and std.mem.eql(u8, path, "/api/v1/projects")) return .{ .route = .projects_list, .id = null };
    if (method == .POST and std.mem.eql(u8, path, "/api/v1/projects")) return .{ .route = .projects_create, .id = null };

    // /api/v1/projects/:id
    const prefix = "/api/v1/projects/";
    if (std.mem.startsWith(u8, path, prefix)) {
        const rest = path[prefix.len..];
        // Find the next '/' if any
        const slash = std.mem.indexOfScalar(u8, rest, '/');
        if (slash == null) {
            // /api/v1/projects/:id
            if (method == .GET) return .{ .route = .projects_get, .id = rest };
            if (method == .DELETE) return .{ .route = .projects_delete, .id = rest };
        } else {
            const id = rest[0..slash.?];
            const tail = rest[slash.? + 1 ..];
            if (method == .GET and std.mem.eql(u8, tail, "chargesheet")) {
                return .{ .route = .projects_chargesheet, .id = id };
            }
        }
    }

    return .{ .route = .not_found, .id = null };
}

test "match returns health for GET /api/v1/health" {
    const m = match(.GET, "/api/v1/health");
    try testing.expectEqual(Route.health, m.route);
    try testing.expect(m.id == null);
}

test "match returns projects_list for GET /api/v1/projects" {
    const m = match(.GET, "/api/v1/projects");
    try testing.expectEqual(Route.projects_list, m.route);
}

test "match extracts id from GET /api/v1/projects/:id" {
    const m = match(.GET, "/api/v1/projects/proj_abc123");
    try testing.expectEqual(Route.projects_get, m.route);
    try testing.expectEqualStrings("proj_abc123", m.id.?);
}

test "match extracts id from DELETE /api/v1/projects/:id" {
    const m = match(.DELETE, "/api/v1/projects/proj_xyz");
    try testing.expectEqual(Route.projects_delete, m.route);
    try testing.expectEqualStrings("proj_xyz", m.id.?);
}

test "match extracts id from GET /api/v1/projects/:id/chargesheet" {
    const m = match(.GET, "/api/v1/projects/proj_abc/chargesheet");
    try testing.expectEqual(Route.projects_chargesheet, m.route);
    try testing.expectEqualStrings("proj_abc", m.id.?);
}

test "match returns cors_preflight for OPTIONS /api/v1/anything" {
    const m = match(.OPTIONS, "/api/v1/anything");
    try testing.expectEqual(Route.cors_preflight, m.route);
}

test "match returns not_found for unknown paths" {

}
```

```zig
const std = @import("std");

/// Generate a `proj_<uuid-v4-with-dashes>` ID. Allocates a 41-byte string.
/// Caller owns the returned slice.
pub fn generateProjectId(gpa: std.mem.Allocator) ![]u8 {
    var bytes: [16]u8 = undefined;
    std.crypto.random.bytes(&bytes);
    // RFC 4122 v4 fixed bits
    bytes[6] = (bytes[6] & 0x0F) | 0x40;
    bytes[8] = (bytes[8] & 0x3F) | 0x80;

    // proj_xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx => 5 + 8 + 1 + 4 + 1 + 4 + 1 + 4 + 1 + 12 = 41
    var buf = try gpa.alloc(u8, 41);
    errdefer gpa.free(buf);
    _ = try std.fmt.bufPrint(buf,
        "proj_{x:0>2}{x:0>2}{x:0>2}{x:0>2}-{x:0>2}{x:0>2}-{x:0>2}{x:0>2}-{x:0>2}{x:0>2}-{x:0>2}{x:0>2}{x:0>2}{x:0>2}{x:0>2}{x:0>2}",
        .{ bytes[0], bytes[1], bytes[2], bytes[3],
           bytes[4], bytes[5], bytes[6], bytes[7],
           bytes[8], bytes[9], bytes[10], bytes[11],
           bytes[12], bytes[13], bytes[14], bytes[15] });
    return buf;
}
```

```zig
const std = @import("std");

/// Layout under `data_dir`:
///   data_dir/<project_id>/chargesheet.pdf
///   data_dir/<project_id>/slices/<filename>.pdf   (Phase 8b)

/// Create the project directory tree. Idempotent.
pub fn createProjectTree(io: std.Io, data_dir: []const u8, project_id: []const u8, gpa: std.mem.Allocator) !void {
    const proj_dir = try std.fs.path.join(gpa, &.{ data_dir, project_id });
    defer gpa.free(proj_dir);
    const slices_dir = try std.fs.path.join(gpa, &.{ proj_dir, "slices" });
    defer gpa.free(slices_dir);
    try std.Io.Dir.cwd().createDirPath(io, proj_dir);
    try std.Io.Dir.cwd().createDirPath(io, slices_dir);
}

/// Write the chargesheet PDF bytes to `<data_dir>/<project_id>/chargesheet.pdf`.
pub fn writeChargesheet(io: std.Io, data_dir: []const u8, project_id: []const u8, bytes: []const u8, gpa: std.mem.Allocator) !void {
    const path = try chargesheetPath(gpa, data_dir, project_id);
    defer gpa.free(path);
    var file = try std.Io.Dir.cwd().createFile(io, path, .{ .truncate = true });
    defer file.close(io);
    try file.writeAll(io, bytes);
}

/// Compute the chargesheet path; caller frees.
pub fn chargesheetPath(gpa: std.mem.Allocator, data_dir: []const u8, project_id: []const u8) ![]u8 {
    return std.fs.path.join(gpa, &.{ data_dir, project_id, "chargesheet.pdf" });
}

/// Remove a project's entire directory tree.
pub fn removeProjectTree(io: std.Io, data_dir: []const u8, project_id: []const u8, gpa: std.mem.Allocator) !void {
    const proj_dir = try std.fs.path.join(gpa, &.{ data_dir, project_id });
    defer gpa.free(proj_dir);
    try std.Io.Dir.cwd().deleteTree(io, proj_dir);
}
```

```zig
pub fn parse(gpa: std.mem.Allocator, boundary: []const u8, body: []const u8) ![]Field {
    var fields = std.ArrayList(Field){};
    errdefer fields.deinit(gpa);

    const delim = try std.fmt.allocPrint(gpa, "--{s}", .{boundary});
    defer gpa.free(delim);

    // Find the first delimiter
    var idx = std.mem.indexOf(u8, body, delim) orelse return error.InvalidMultipart;

    while (idx < body.len) {
        // After "--BOUND", expect either "\r\n" (next part) or "--" (end)
        const after = idx + delim.len;
        if (after + 2 > body.len) return error.InvalidMultipart;
        if (body[after] == '-' and body[after + 1] == '-') {
            // closing boundary "--BOUND--"
            return try fields.toOwnedSlice(gpa);
        }
        if (body[after] != '\r' or body[after + 1] != '\n') return error.InvalidMultipart;

        // Read headers until \r\n\r\n
        const headers_start = after + 2;
        const headers_end = std.mem.indexOfPos(u8, body, headers_start, "\r\n\r\n") orelse return error.InvalidMultipart;
        const headers = body[headers_start..headers_end];
        const body_start = headers_end + 4;

        // Find next delimiter
        const next_idx = std.mem.indexOfPos(u8, body, body_start, delim) orelse return error.InvalidMultipart;
        // Body is bytes from body_start up to but not including "\r\n--BOUND"
        if (next_idx < 2 or !std.mem.eql(u8, body[next_idx - 2 .. next_idx], "\r\n")) return error.InvalidMultipart;
        const body_end = next_idx - 2;
        const field_body = body[body_start..body_end];

        // Parse Content-Disposition for name= and filename=
        var name: ?[]const u8 = null;
        var filename: ?[]const u8 = null;
        var lit = std.mem.splitSequence(u8, headers, "\r\n");
        while (lit.next()) |line| {
            if (std.ascii.startsWithIgnoreCase(line, "Content-Disposition:")) {
                name = extractQuoted(line, "name=");
                filename = extractQuoted(line, "filename=");
            }
        }
        if (name == null) return error.InvalidMultipart;

        try fields.append(gpa, .{ .name = name.?, .filename = filename, .body = field_body });
        idx = next_idx;
    }

    return error.InvalidMultipart;
}

fn extractQuoted(line: []const u8, key: []const u8) ?[]const u8 {
    const start = std.mem.indexOf(u8, line, key) orelse return null;
    const after_key = start + key.len;
    if (after_key >= line.len or line[after_key] != '"') return null;
    const value_start = after_key + 1;
    const value_end = std.mem.indexOfScalarPos(u8, line, value_start, '"') orelse return null;
    return line[value_start..value_end];
}
```

```zig
/// Escape a string for JSON. Output to `w`. Handles `"`, `\`, control chars.
pub fn writeJsonString(w: *std.Io.Writer, s: []const u8) !void {
    try w.writeAll("\"");
    for (s) |c| {
        switch (c) {
            '"' => try w.writeAll("\\\""),
            '\\' => try w.writeAll("\\\\"),
            '\n' => try w.writeAll("\\n"),
            '\r' => try w.writeAll("\\r"),
            '\t' => try w.writeAll("\\t"),
            0...0x1F => try w.print("\\u{x:0>4}", .{c}),
            else => try w.writeByte(c),
        }
    }
    try w.writeAll("\"");
}
```

```zig
pub const CreateError = error{

    InvalidName,
    InvalidDescription,
    InvalidPdf,
    NameConflict,
    OutOfMemory,
    DbError,
    PdfError,
    IoError,

pub const CreateResult = struct {
    project: projects_mod.Project,
    /// Caller must call `project.deinit(gpa)` when done with the result.
    pub fn deinit(self: *CreateResult, gpa: std.mem.Allocator) void {
        self.project.deinit(gpa);
    }

/// Validate inputs, write the chargesheet to disk, insert the project row,
/// return the created Project ready for JSON serialization.
pub fn handleProjectsCreate(
    gpa: std.mem.Allocator,
    io: std.Io,
    db: *Db,
    mupdf_ctx: *mupdf.Context,
    data_dir: []const u8,
    boundary: []const u8,
    body: []const u8,
) CreateError!CreateResult {
    const fields = multipart.parse(gpa, boundary, body) catch return error.InvalidRequest;
    defer gpa.free(fields);

    var name: ?[]const u8 = null;
    var description: ?[]const u8 = null;
    var chargesheet_bytes: ?[]const u8 = null;
    var chargesheet_filename: ?[]const u8 = null;

    for (fields) |f| {
        if (std.mem.eql(u8, f.name, "name")) name = f.body
        else if (std.mem.eql(u8, f.name, "description")) description = f.body
        else if (std.mem.eql(u8, f.name, "chargesheet")) {
            chargesheet_bytes = f.body;
            chargesheet_filename = f.filename;
        }
    }

    const name_trimmed = if (name) |n| std.mem.trim(u8, n, " \t\r\n") else return error.InvalidName;
    if (name_trimmed.len == 0 or name_trimmed.len > 200) return error.InvalidName;
    if (description) |d| {
        if (d.len > 2000) return error.InvalidDescription;
    }
    if (chargesheet_bytes == null or chargesheet_bytes.?.len == 0) return error.InvalidPdf;

    // Uniqueness check (DB UNIQUE on name still enforces; this is friendlier)
    if (projects_mod.existsByName(db, name_trimmed) catch return error.DbError) return error.NameConflict;

    const id = ids.generateProjectId(gpa) catch return error.OutOfMemory;
    errdefer gpa.free(id);

    project_dir.createProjectTree(io, data_dir, id, gpa) catch return error.IoError;
    errdefer project_dir.removeProjectTree(io, data_dir, id, gpa) catch {};

    project_dir.writeChargesheet(io, data_dir, id, chargesheet_bytes.?, gpa) catch return error.IoError;

    // Use mupdf-zig to count pages from the file we just wrote.
    const path = project_dir.chargesheetPath(gpa, data_dir, id) catch return error.OutOfMemory;
    defer gpa.free(path);
    const path_z = gpa.dupeZ(u8, path) catch return error.OutOfMemory;
    defer gpa.free(path_z);

    var doc = mupdf.Document.open(mupdf_ctx, path_z) catch return error.InvalidPdf;
    defer doc.deinit();
    const page_count = doc.pageCount() catch return error.PdfError;

    const now = db_mod.nowIso8601(gpa) catch return error.OutOfMemory;
    defer gpa.free(now);

    // Build the row, insert via projects.insert.
    const filename = if (chargesheet_filename) |f| f else "chargesheet.pdf";
    const description_owned: ?[]const u8 = if (description) |d| (gpa.dupe(u8, d) catch return error.OutOfMemory) else null;

    const row: projects_mod.Project = .{
        .id = id,
        .name = gpa.dupe(u8, name_trimmed) catch return error.OutOfMemory,
        .description = description_owned,
        .created_at = gpa.dupe(u8, now) catch return error.OutOfMemory,
        .last_opened_at = gpa.dupe(u8, now) catch return error.OutOfMemory,
        .chargesheet_filename = gpa.dupe(u8, filename) catch return error.OutOfMemory,
        .chargesheet_page_count = page_count,
        .chargesheet_size_bytes = chargesheet_bytes.?.len,
    };

    projects_mod.insert(db, gpa, row) catch |err| switch (err) {
        error.UniqueViolation => return error.NameConflict,
        else => return error.DbError,
    };

    return .{ .project = row };
}
```

### Task 7: `GET /api/v1/projects/:id/chargesheet`

Stream the uploaded PDF back to the client. Read the file from disk, write to response with `Content-Type: application/pdf`.

**Files:**

- [ ] **Step 1: Add `handleProjectsChargesheet`**

```zig
pub const ChargesheetError = error{ NotFound, IoError, OutOfMemory };

pub fn handleProjectsChargesheet(
    gpa: std.mem.Allocator,
    io: std.Io,
    db: *Db,
    data_dir: []const u8,
    id: []const u8,
    out: *std.Io.Writer,
) ChargesheetError!struct { filename: []const u8 } {
    var maybe = projects_mod.getById(db, gpa, id) catch return error.IoError;
    var project = maybe orelse return error.NotFound;
    defer project.deinit(gpa);

    const path = project_dir.chargesheetPath(gpa, data_dir, id) catch return error.OutOfMemory;
    defer gpa.free(path);

    var file = std.Io.Dir.cwd().openFile(io, path, .{ .mode = .read_only }) catch return error.NotFound;
    defer file.close(io);

    var buf: [64 * 1024]u8 = undefined;
    var offset: u64 = 0;
    while (true) {
        const n = file.readPositionalAll(io, &buf, offset) catch return error.IoError;
        if (n == 0) break;
        out.writeAll(buf[0..n]) catch return error.IoError;
        offset += n;
        if (n < buf.len) break;
    }

    // We can't return the filename via the caller-owned project after returning, so
    // we duplicate the filename to gpa and the caller frees. Simpler: the server
    // sets Content-Disposition before this call using the filename it can fetch
    // separately. Adjust the API as the implementer prefers — the OUTCOME is that
    // the response has Content-Type: application/pdf and Content-Disposition with
    // the project's filename.
    return .{ .filename = "" };
}
```

The exact API for streaming a file body via `std.http.Server.Request.respondStreaming(...)` is at `lib/std/http/Server.zig:412`. The implementer reads that file and adapts the loop above so the writer is the response's body stream.

- Bad multipart → 400 error JSON.
- Disk write fails mid-upload → DB row not inserted (the order in `handleProjectsCreate` is: validate → write file → count pages → insert row, with `errdefer removeProjectTree` cleaning up partial files).
- DB constraint violation → 409 or 500 depending on cause.

**5. Epistemological humility (Gate 5):**
- The HTTP stack relies on Zig 0.16 stdlib (`std.http.Server`, accept-loop in `std.Io.net`). If the stdlib API differs from what `lib/std/http/test.zig` shows, the implementer adapts — that's explicit in Task 1 Step 1.
- The multipart parser handles only the subset the UI sends. RFC 7578 conformance is partial. Out of scope to fix until a real-world client breaks it.
- No backpressure on uploads — we read the whole body into RAM. A 1GB PDF would OOM the daemon. Mock does the same; matching the mock's behavior is the goal here, not surpassing it. Phase 8c can add streaming if needed.
- No signal-handler shutdown in Phase 8a — `kill <pid>` is the only way to stop the daemon cleanly. The plan flags this as Phase 8c work in Task 1 Step 5.
